# app-cleaner

## Setup guide

### Step 1: create `/usr/local/bin` directory

```shell
sudo mkdir -p /usr/local/bin
sudo chown ${USER} /usr/local/bin
```

### Step 2: download [cleaner.sh](./cleaner.sh)

```shell
cd /usr/local/bin
download [cleaner.sh](./cleaner.sh)
chmod +x /usr/local/bin/cleaner.sh
```
